Introdução ao Banco de Dados

Conceitos Básicos, Modelos e Tecnologias

O que é um Banco de Dados?

Um banco de dados é uma coleção organizada de dados, que pode ser facilmente acessada, gerenciada e atualizada.

Tipos de Banco de Dados

  • Relacional (SQL): Baseado em tabelas, com SQL como linguagem de consulta.
  • Não Relacional (NoSQL): Usado em dados não estruturados ou semi-estruturados.

Modelos de Banco de Dados

  • Modelo Relacional: Usa tabelas com linhas e colunas.
  • Modelo de Documento: Dados armazenados em documentos JSON ou BSON.
  • Modelo de Grafos: Dados representados por nós e arestas.
  • Modelo Chave-Valor: Armazenamento de pares chave-valor.

Sistema de Gerenciamento de Banco de Dados (SGBD)

O SGBD é responsável por gerenciar o banco de dados e fornecer uma interface para interagir com os dados.

  • Exemplos de SGBD Relacional: MySQL, PostgreSQL.
  • Exemplos de SGBD NoSQL: MongoDB, Redis.

SQL vs NoSQL

SQL: Bancos de dados relacionais, com estrutura fixa e suporte a transações.

NoSQL: Bancos de dados não relacionais, mais flexíveis, adequados para grandes volumes de dados.

Estrutura de Banco de Dados Relacional

  • Tabelas: Estruturas compostas por linhas e colunas.
  • Chaves Primárias: Identificadores únicos de cada linha.
  • Chaves Estrangeiras: Relacionam dados de uma tabela com outra.

Operações CRUD

  • Create: Inserir novos dados.
  • Read: Consultar dados existentes.
  • Update: Atualizar dados existentes.
  • Delete: Excluir dados.

Conclusão

Os bancos de dados são fundamentais para organizar, armazenar e gerenciar grandes volumes de dados em sistemas modernos. São usados em diversas áreas, como e-commerce, redes sociais, bancos e muito mais.
